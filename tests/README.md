# Tests This directory contains lightweight tests and validation scripts for the repository. Scripts here are intended to run on a host machine or in CI and should not assume a rooted Android environment unless explicitly documented. ## Test Files - `shellcheck_smoke.sh`: runs `shellcheck` against all checked-in shell scripts and reports violations. - `path_sanity.sh`: checks for leftover lowercase username references and broken bare-home paths. ## Running Tests From the repo root: ```bash bash tests/shellcheck_smoke.sh bash tests/path_sanity.sh ./validate.sh ``` Each script exits non-zero on failure, so they can be chained or wired into CI.
