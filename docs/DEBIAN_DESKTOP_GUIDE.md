# 🐧 Debian Desktop Guide ## Scope This guide explains how the Debian guest desktop is launched from Termux, and how to manage the XFCE session for `Ruusian5`. ## Launch Options - Dashboard: `agy` → Option 1 - Direct: ```bash bash ~/Systemless-Host-Termux-SU/scripts/startxfce4_chrootDebian.sh ``` ## Session Details - Desktop: XFCE - User: `Ruusian5` - Display: `:0` - Runtime: `/run/user/1000` - Audio: `PULSE_SERVER` forwarded from host ## Troubleshooting - D-Bus race: `user-session.sh` runs `dbus-launch --sh-syntax` before starting XFCE. - Black screen: disable compositing or switch `LIBGL_ALWAYS_HW=0`. - Clipboard not syncing: ensure `clipboard-sync.sh` is running on host. ## Related Docs - [docs/TERMUX_X11_GUIDE.md](./TERMUX_X11_GUIDE.md) - [ARCHITECTURE.md](../ARCHITECTURE.md)
