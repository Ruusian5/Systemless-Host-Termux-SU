# Tools This directory contains helper utilities maintained as part of the repo. Tools here should be safe to run on a host workstation and should not modify state outside the repository without explicit confirmation. ## Contents - `path_rewrite.sh`: rewrites lowercase guest-username paths to the canonical form. - `offline_pack.sh`: archives the chroot directory into a zstd snapshot. ## Usage Examples ```bash bash tools/path_rewrite.sh bash tools/offline_pack.sh ```
